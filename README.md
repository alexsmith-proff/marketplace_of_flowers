## Description
Marketplace of flowers 

## Stack
NestJS/Next.js/TypeScript, GraphQL, PostgreSQL

## Installation

```bash
$ cd backend
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
