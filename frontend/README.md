## Description

Frontend marketplace of flowers

## Stack
Next.js/TypeScript, GraphQL

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev
```

Follow the link http://localhost:3000 in a browser
To access the admin panel follow the link http://localhost:3000/admin
