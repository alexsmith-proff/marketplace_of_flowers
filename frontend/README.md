## Description

Frontend marketplace of flowers

## Stack
Next.js/TypeScript, GraphQL, REDUX

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# build mode
$ npm run build

# start mode
$ npm run startd
```
